Codebeispiel::SinglePageApplikation (SPA)

Dieses Projekt dient als Codebeispiel für eine SPA (SinglePageApplikation)

Diese Applikation

Mit der Eingabe Ihres Namens in untere Formular und dem Klick auf Senden wird Ihre Eingabe an den Server übermittelt. Dieser sendet eine Antwort und diese löst einen Alert aus mit einer Nachricht.

Möglicher UseCase für eine SPA

Für einen Warenkorb (eine Unterseite) werden sämtliche Logiken, wie z.Bsp. Produktauswahl, Berechnungen oder vom Server gelieferte Produktlisten filtern, werden via JavaScript Clientseitig abgearbeitet und zu persistierende Daten im JSON-Format mit mit einem POST-Request an den Server geschickt.

Ein Vorteil unter vielen: Dieses Vorgehen entlasten den Server und die Kommunikation über das Internet und der Benutzer auf der Client-Seite hat besseres Feeling beim Benutzen der App.

Eingesetzte Werkzeuge und Techniken

Grundsätzlich läuft die Applikation Serverseitig mit Node.js (V 12.15.0 LTS).
